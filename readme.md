# React App Template

Easier setup for react apps

## Installation

```bash
npm install
```

## Purpose

Setup:

- clean index.html
- clean index.js
- clean index.css
- general App.js
- babelrc file

## Scripts

```javascript
    "dev": "parcel src/index.html",
    "build": "parcel build src/index.html",
    "start": "npm run build && live-server dist"
```
